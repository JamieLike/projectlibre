ProjectLibre is the updated OpenProj.
The major improvements includes:
- Compatibility with Microsoft Project 2010
- New Ribbon User Interface
- PDF feature
- Print, print preview improvements
- Major bug fixes and more


Mac OS X notes:
ProjectLibre requires Mac OS X 10.5.7 or later.
- Versions 10.5: download projectlibre-1.5.3.dmg. Update to 10.5.7+. Make sure java updates have been installed (http://support.apple.com/kb/DL848). Then select Java 6 as default in /Applications/Utility/Java Preferences.
- Versions 10.6, 10.7, 10.8 with Java 6: download projectlibre-1.5.3.dmg.
- Versions 10.7.3+, 10.8 with Java7: download projectlibre-1.5.3_java7.dmg

Linux notes:
- Fedora/RedHat Enterprise/Oracle/Centos, openSUSE/SLES/SLED, Mageia/Mandriva/PCLinuxOS: download projectlibre-1.5.3-1.rpm
- Ubuntu/Mint/Debian: download projectlibre-1.5.3-1.deb
- Slackware and distributions not using rpm or deb: download projectlibre-1.5.3.tar.gz
